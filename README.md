# ddu-source-jumplist

jumplist source for ddu.vim

## Required
denops.vim  
https://github.com/vim-denops/denops.vim

ddu.vim  
https://github.com/Shougo/ddu.vim

ddu-kind-file  
https://github.com/Shougo/ddu-kind-file

## Configuration
```vim
call ddu#start(#{ sources: [#{ name: 'jumplist' }] })
```
